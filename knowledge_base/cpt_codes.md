# AlphaMed Clinic - CPT Code Reference Guide

## CPT 99212
CPT Code 99212 represents a brief outpatient office visit for the evaluation and management of an established patient. 
Typically involves a minor problem or straightforward medical decision-making, lasting 10 to 19 minutes.

## CPT 99213
CPT Code 99213 represents a standard mid-level outpatient office visit for the evaluation and management of an established patient. 
Typically involves a low-complexity medical decision-making process, lasting 15 to 29 minutes.

## CPT 99214
CPT Code 99214 represents a comprehensive mid-to-high level outpatient office visit for the evaluation and management of an established patient. 
Typically involves moderate-complexity medical decision-making, lasting 30 to 39 minutes.

## CPT 36415
CPT Code 36415 represents a routine venipuncture procedure, which is the collection of a venous blood sample by draw (blood draw/lab draw). 
It is standard for lab and diagnostic testing.

## CPT 72148
CPT Code 72148 represents a magnetic resonance imaging (MRI) scan of the lumbar spine (lower back) without contrast material. 
It is commonly used to diagnose herniated discs, sciatica, spinal stenosis, or persistent lower back pain.

## CPT 94640
CPT Code 94640 represents an inhalation treatment (such as a nebulizer therapy) for acute airway obstruction. 
It is commonly administered for patients suffering from asthma flares, COPD, or acute wheezing.
